Notes

  • Uses Places API (New) Text Search — enable "Places API (New)" in Google Cloud Console and allow your domain (and localhost/file origins while testing) in key restrictions.
  • Results are de-duplicated across searches; a business matching several searches gets multiple category tags.
  • Each city fetch makes 11 requests. Review current Google Maps Platform pricing, quotas, key restrictions, and billing controls before running a batch.
  • Review results before committing — drop anything that clearly isn't a real service provider.
